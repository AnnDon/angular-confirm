Angular-confirm

Confirmation modal dialog for AngularJS

Download this project as a .zip file Download this project as a tar.gz file

Usage

This module depends on Angular-UI bootstrap for its $modal service and can be used in two different ways


As a directive

The confirm directive can be placed on anything that also has ngClick

<button type="button" ng-click="delete()" confirm="Are you sure, {{name}}?">Delete</button>

The confirmIf directive can also be used if you only want the confirmation dialog to show if the expression supplied is true

<button type="button" ng-click="delete()" confirm-if="checked" confirm="Are you sure, {{name}}?">Delete</button>

This button will always show a dialog if the checkbox is checked.
The "OK" button in the dialog will execute the ngClick expression on the button.
Enter your name in the text box to observe it shows up in the confirmation text.

As a Service

$confirm(data, success, [settings])

The $confirm service is available for you to show confirmation dialogs. This allows you to show confirmations without having to have an HTML element tied to the action.

Param Type Details
data Object Data to be used in the modal HTML
success Function The function that will be executed when the user confirms the action
settings Object Optional. Additional settings to be passed to $modal 
.controller("Controller", function($scope, $confirm) {
  $scope.delete = function() {
    $confirm({text: 'Are you sure you want to delete?'}, function() {
      $scope.deleted = 'Deleted';
    });
  }
})

Settings

You can inject and modify $confirmModalDefaults to change the default settings that get passed to the $modal service.

.run(function($confirmModalDefaults) {
  $confirmModalDefaults.templateUrl = 'path/to/your/template';
})

You may also pass settings that will override the default settings when using $confirm directly.

var settings = {
  templateUrl: 'path/to/your/template'
};

$confirm({}, function() {
...
}, settings);

If you provide your own template or templateUrl you may use any data passed to $confirm by referencing it with data.. The default template expects a "text" key to show the confirmation text.

var settings = {
  template: 'Hello {{data.name}}!'
};

$confirm({name: "Sally"}, function() {
...
}, settings);
